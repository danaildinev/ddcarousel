Centered slide

Example with centered slide. Currently only one slide per swipe is possible even though there are multiple slides per page. Active slide is the centered one.

Auto height + one item per page

Change carousel height automatically on slide change based on the highest visible slide. Also showing the new feature to change one item per swipe.

Responsive

Try resizing the page to see different options based on viewport. You can also see the new 'fullWidth' option enabled below 768px width.

Vertical orientation + starting page

This examples shows vertical orientation of carousel with two slides per page and starting page set to 3.

Lazy load images (new)

Lazy loading of images in carousel. This demo will lazy load images on changing active slides and also preload the next 2 images (total of 4 images loaded).

Autoplay

Autoplay feature with speed set to 3 seconds, enabled pause on hover/touch and progress bar (enabled by default). Use buttons to toggle on/off.

Disabled touch drag

In this example, touch is disabled but mouse dragging still works.

Custom navigation labels

Use custom labels for navigation links.

Custom animation config

In this example some animation properties are changed.

touchSwipeThreshold is changed to 1, so the user can swipe easier than default. touchMaxSlideDist is also set to high value, so that user can swipe longer before reaching the max swiping distance. slideChangeDuration and swipeSmooth is increased for smoother experience.

Keyboard navigation (new)

Use arrow keys on your keyboard to change slides.
